# emkt-generator

## emkt-generator?

> emkt-generator is a abbreviation to *email marketing generator*.

## Why?

> Beause i'm tired to write css inline for email platforms.

## What it does?

> This project takes the HTML file with CSS and transform into a inline style HTML.

----
## Usage
1. Create folder **src** in project root
2. Create a folder to your email: **src/project-zeta**
3. Create your **html** and **css** code inside your project folder (project-zeta in case)
4. When its done, just edit the project variable **folder** inside **gulpfile.js** and run the command `gulp` in root
